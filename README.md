For 4th Assignment of Tutedude Python Course

Assignment 4
Module 5 – Files, Exceptions and Errors in Python

Task 1 : Read a File and Handle Errors – 
In this task, a file named ‘sample.txt’ is opened in read mode and its contents are read line by line. While opening the file, the error is handled if the file does not exist.
 
 
Task 2 : Write and Append Data to a File –
In this task, a file named ‘output.txt’ is opened in write mode. An input is accepted from the user and written to the file. After this additional input is taken from the user and appended to the file. Finally entire content of the file is displayed.
